## 🏙️ Project CivicLens

A futuristic civic issue tracking and resolution platform built for hackathons.

Project Polis helps citizens report, track, and visualize civic issues on an interactive map while enabling real-time status updates through a Kanban workflow and analytics dashboard.

## ✨ Features

### 🗺️ Interactive Civic Map
- Click anywhere to report issues
- Category-based colored pins
- Zoom and pan interactions
- Heatmap mode for issue density visualization

---

### 📋 Kanban Workflow System
- Drag & drop issue cards
- Three stages: New → In Progress → Resolved
- Upvote system to prioritize issues
- Auto-sorting based on community votes

---

### 📊 Analytics Dashboard
- Real-time issue trends
- Category distribution insights
- Resolution rate tracking
- Data-driven civic insights

---

### 🔄 Real-Time Sync System
- Map, Kanban, and Analytics share a unified state
- Updates reflect instantly across all components
- LocalStorage persistence (no backend required)

---

### 🎨 Premium UI/UX
- Glassmorphism (Apple-style UI)
- Dark cyber aesthetic theme
- Smooth animations and transitions
- Fully responsive design

## ⚙️ How It Works

1. Users click on the map to report civic issues  
2. Issues are stored in a shared state (localStorage)  
3. Issues appear simultaneously on:
   - Map (as pins)
   - Kanban board (as cards)
   - Analytics dashboard (as data points)
4. Users can upvote and track resolution progress
